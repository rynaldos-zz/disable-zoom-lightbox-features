== Description ==

This plugin lets you disable / enable the new product gallery zoom / lightbox features (or both), in WooCommerce 3.0.


== Installation ==

1. Download the plugin & install it to your `wp-content/plugins` folder (or use the Plugins menu through the WordPress Administration section)
2. Activate the plugin
3. Navigate to **WooCommerce > Settings > Products > Product gallery options**.
4. Select your option/s.
5. Save and enjoy!
